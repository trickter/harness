# Escape Divergence

Summarize the failed path before suggesting a new one.

1. State why the current hypothesis failed.
2. Produce at least three alternative hypotheses.
3. Select a strategy that differs materially from the frozen path.

Read [references/escape-output.md](references/escape-output.md) for the output shape.
