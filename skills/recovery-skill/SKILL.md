# Recovery Skill

Provide structured recovery instructions when a run has failed or needs to backtrack to a clean snapshot.

1. Assess the current changes, git status, and failure reports.
2. Formulate advice on whether to keep or revert changed artifacts.
3. Outline specific manual actions or commands to run to recover the workspace.

Read [references/recovery-report.md](references/recovery-report.md) for output formatting.
