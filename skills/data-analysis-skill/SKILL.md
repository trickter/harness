# Data Analysis Skill

Perform exploratory data analysis and record findings on dataset quality, missing values, duplicates, and outliers.

1. Assess target dataset structure and values.
2. Outline findings on missing values, outliers, and data distributions.
3. Establish confidence statements and quality gates for downstream modeling.

Read [references/data-analysis-output.md](references/data-analysis-output.md) for report schema.
