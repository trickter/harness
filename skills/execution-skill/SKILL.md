# Execution

Execute one supervised action only.

1. Restate the allowed operation and artifacts.
2. Refuse an action that expands scope or hides multiple material edits.
3. Return changed artifacts, commands, and new information for verification.

Read [references/action-result.md](references/action-result.md) for the action result shape.
